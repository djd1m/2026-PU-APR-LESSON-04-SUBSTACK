## 2. PRODUCT & CUSTOMERS

### A. One-Liner

> **Substack** — это WordPress для рассылок + встроенные платные подписки с нулевой стоимостью запуска

### B. Problem Statement

| Измерение | Без Substack | С Substack | Улучшение |
|-----------|-------------|------------|-----------|
| **Время запуска** | 2–4 недели (домен + CMS + ESP + платежи) | 5–15 минут | ~100x быстрее |
| **Стоимость** | $50–200/мес фикс (хостинг + Mailchimp) | $0 до первого заработка | Нулевой риск |
| **Усилия** | Нужен разработчик для интеграций | Без технических знаний | Самостоятельно |
| **Аудитория** | Алгоритмы (FB reach 2–5%) | Прямой email, open rate 45%+ | 2–3x engagement |
| **Монетизация** | Отдельно Stripe + подписочная логика | Один переключатель | Месяцы → 1 клик |

### C. 10x Improvement

- **vs WordPress + Mailchimp:** нулевая инфраструктура, единая платформа, оплата только с дохода
- **vs Medium:** владение аудиторией, прямая монетизация, нет алгоритмической зависимости
- **vs Ghost:** нет DevOps-нагрузки, сетевые эффекты, конвергенция форматов (текст + аудио + видео)

## 5. GROWTH ENGINE

### A. Primary Growth Loop

**Content-Led + Network Effects** (как Substack):

```
Автор публикует → Бесплатные подписываются по email
    → Часть конвертится в paid → Автор зарабатывает → Автор привлекает коллег
        → Рекомендации показывают автора существующим пользователям
            → Кросс-опыление → Сеть плотнеет → Платформа растёт
```

У Substack: рекомендации дают **25% всех платных подписок**, конверсия в paid **3x выше** для пользователей внутри сети.

### B. Top-3 каналов для РФ

| # | Канал | CAC | Timing |
|---|-------|-----|--------|
| 1 | Дзен/VK cross-posting + партнёрства с авторами | 150–300 ₽ | M1–3 |
| 2 | SEO/Yandex (платформенный домен + архивы) | Органический | M3–6 |
| 3 | VK Communities + реферальная программа | 200–400 ₽ | M4–12 |

### C. Moats

| Moat | Стратегия |
|------|----------|
| Сетевые эффекты | Рекомендации с первого дня, "читатели этого автора также читают..." |
| Регуляторный | 152-ФЗ + реестр + платежи = западные не могут конкурировать |
| Контентная библиотека | Yandex-индексированный архив — SEO juice остаётся на домене |
| Платёжный lock-in | Сохранённые карты Mir, подписки через СБП |
| Data moat | Паттерны потребления русскоязычного контента |

### D. Специфика России

- **VK/Дзен интеграция** — кросс-постинг и импорт аудитории
- **Платёжный стек:** CloudPayments (recurring) + СБП (low-cost) + YooKassa (fallback)
- **Yandex SEO** (60%+ поискового рынка в РФ)
- **Ниши с высокой монетизацией:** финансы/инвестиции, независимая журналистика, профзнания (юристы, бухгалтеры), бизнес/стартапы
